# Анализ пользовательского поведения и конверсии сайта «СберАвтоподписка»

Анализ данных веб-аналитики сервиса долгосрочной аренды автомобилей. Исследование посвящено выявлению факторов, влияющих на конверсию посетителей в целевые действия (оформление заявки, заказ звонка и др.), и формированию рекомендаций по оптимизации рекламных кампаний.

---

## О проекте

**Задача:** Проанализировать поведение пользователей на сайте «СберАвтоподписка» на основе данных из Google Analytics и ответить на ключевые вопросы продуктовой команды:

- Из каких источников приходит самый целевой трафик?
- Какие автомобили пользуются наибольшим спросом?
- Стоит ли увеличивать присутствие в социальных сетях?

**Что было сделано:**

- Объединение и очистка двух датасетов (события и визиты) общим объемом ~15 млн записей.
- Создание новых признаков: тип трафика (органический/платный), тип устройства (мобильное/десктопное), города присутствия, извлечение марки и модели автомобиля из URL.
- Проверка трех статистических гипотез о влиянии трафика, устройств и геолокации на конверсию.
- Формулировка практических рекомендаций для бизнеса.

## Основные выводы

### 1. Проверка гипотез

| Гипотеза | Результат | Вывод |
|----------|-----------|-------|
| Органический трафик = Платный по CR | Отвергнута | Органический трафик дает БОЛЬШУЮ конверсию |
| Мобильные = Десктопные по CR | Подтверждена | Статистически значимого отличия нет |
| Города присутствия = Другие регионы по CR | Подтверждена | Статистически значимого отличия нет |
| Реклама в соцсетях = Другие каналы по CR | Отвергнута | Реклама в соцсетях дает МЕНЬШУЮ конверсию |


### 2. Источники целевого трафика:
   - **Бренды устройств**: Samsung, OPPO, Vivo, Apple
   - **Браузеры**: Mozilla Firefox, Microsoft Edge, Yandex Browser
   - **Локация**: Татарстан, Краснодарский Край, Чечня, Москва, Башкортостан
   - **Соцсети:** конверсия 2.1% против 3.3% у других каналов

### 3. Автомобили

- **Автопроизводитель:** volkswagen, toyota, kia, lada-vaz, skoda
- **Модели:** toyota camri, volkswagen polo, kia rio, kia seltos, skoda rapid (бюджетные автомобили)

### 4. Временные паттерны

- Конверсия выше в начале рабочей недели

### 5. Рекомендации

1. Инвестировать в органический трафик (SEO, контент-маркетинг)
2. Не увеличивать бюджет на рекламу в соцсетях
3. Продвигать бюджетные модели: Toyota Camry, Volkswagen Polo, Kia Rio, Skoda Rapid
4. Усилить рекламную активность в начале недели
